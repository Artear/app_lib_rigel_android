MainFragment()

It is the base fragment for this library. Each section in NavigationActivity has an MainFragment witch control the update of the action bar when use a ActionBarFragment in the vertical navigation.

This fragment is a unique instance and save the NavigationSection if is destroyed.

Launch in your childFragmentManager each fragment. First attach the fragment provided by NavigationSection. And if that fragment needs launch another call to the parent and execute launchFragment.
